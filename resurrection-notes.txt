* aktywacja WSL Windows Subsystem for Linux w Windows
wykonujemy instrukcje zawarte tutaj:
https://docs.microsoft.com/en-us/windows/wsl/install-win10
--> uwaga! aktywujemy WSL 2!

* instalujemy Ubuntu-18.04 LTS z Microsoft Store

* aktualizacja repozytoriów, instalacja python i python-pip
sudo apt-get update
sudo apt-get install python
sudo apt-get install python-pip

* instalacja virtualenvwrapper za pomocą pip
$ pip install virtualenv

* tworzenie virtualenv o nazwie 'venv-btsearch' --> virtualenv to zamknięte środowisko, w którym zainstalowane zostaną pakiety niezbędne do wystartowania projektu - pakiety te będą zainstalowane tylko lokalnie wewnątrz virtualenv, nie będą dostępne dla reszty systemu
$ virtualenv venv-btsearch

* tworzenie folderu projektowego w folderze domowym /home/USERNAME
$ cd ~
$ mkdir dev
$ cd dev

* klonowanie repo z guthub
$ git clone git@github.com:adlorenz/btsearch.git

* wejście do sklonowanego z repozytorium folderu projektowego
$ cd btsearch
--> aktualna ścieżka to /home/USERNAME/dev/btsearch/

* aktywacja virtualenv
$ . ~/venv-btsearch/bin/activate

--> w linii poleceń znajdzie się przedrostek wskazujący na aktywowany virtualenv
(venv-btsearch) 00:08 evad@LATITUDE7380 btsearch (develop) $
